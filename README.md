# The Human Zoo 🦒

> A living exhibition of human behavior.

This is not a museum.  
This is a **zoo** — where the animals are us.

Every exhibit documents a real, recurring pattern of human psychology, behavior, or coping mechanism. No judgment. Just observation.

**Warning:** You may recognize yourself in several cages.

### How to Submit a New Exhibit
1. Create a new file in the `exhibits/` folder
2. Use the [exhibit template](./new-exhibit-template.md)
3. Open a Pull Request
